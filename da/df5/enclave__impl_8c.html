void notify_tag_advance_grant ( enclave_t *  e,
tag_t  tag 
)

Notify a tag advance grant (TAG) message to the specified federate. Do not notify it if a previously sent PTAG was greater or if a previously sent TAG was greater or equal.

This function will keep a record of this TAG in the federate's last_granted field.

This function assumes that the caller holds the mutex lock.
